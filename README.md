## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- body-parser
- cookie-session
- method-override

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
- Got to `http://localhost:8080/` in your browser.
